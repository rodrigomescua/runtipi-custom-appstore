# MurkPyRSS

MurkPyRSS is a modern RSS feed generator designed to scrape and convert websites into RSS feeds. It is the successor to MurkRSS, built with better performance and more robust scraping capabilities.

## Features

- **Hybrid Scraping**: Combine automated detection with manual selectors.
- **Visual Selector**: Easily point and click to define the data you want to extract.
- **FlareSolverr Integration**: Bypass Cloudflare and other anti-bot protections.
- **Persistent Data**: All your feeds and configurations are saved locally.
- **Custom Intervals**: Control how often each site is scraped.
