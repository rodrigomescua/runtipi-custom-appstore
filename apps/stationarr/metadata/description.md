# Stationarr

Self-hosted IPTV playlist editor, EPG matcher, and companion for the *arr ecosystem.

**Features:**
- Multi-user accounts with JWT authentication
- Import M3U playlists via file upload or server-side URL
- Visual channel editor with drag & drop reordering
- Automatic EPG matching by channel ID and display name
- Logo auto-matching and logo browser (10,000+ logos)
- Multiple EPG (XMLTV) sources with URL fetch or file upload
- Hosted M3U and XMLTV output URLs per playlist
- Xtream API output format support
- Admin panel with user management
- Optional iptv-org/epg scraper integration via Docker sidecar
- SQLite database (zero config)
- Docker deployment ready

**Important:** Stationarr is a playlist and EPG management tool only. Users are responsible for ensuring all playlists, EPG sources, and streams used are legally obtained and compliant with applicable laws.
