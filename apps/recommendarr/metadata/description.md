# Recommendarr

An LLM-driven recommendation system for personalized TV show and movie suggestions.

## Features

- **AI-Powered Recommendations**: Get personalized suggestions based on your existing library using advanced AI models
- **Multi-Service Integration**: Connects to Sonarr, Radarr, Plex, Jellyfin, Tautulli, and Trakt
- **Flexible AI Support**: Works with OpenAI, local models (Ollama/LM Studio), or any OpenAI-compatible API
- **Multi-User Support**: Admin users can manage additional users with customizable permissions
- **Watch History Analysis**: Analyzes your watch history for better, more personalized recommendations
- **TMDB Integration**: Optional TMDB integration for detailed media information and posters
- **Dark/Light Mode**: Toggle between themes based on your preference
- **PWA Support**: Install as an application on your phone or desktop

## Default Credentials

- **Username**: `admin`
- **Password**: `1234`

⚠️ **Important**: Change your password immediately after first login for security reasons.

## Configuration

After installation, you'll need to:
1. Connect your media services (Sonarr, Radarr, Plex, Jellyfin, Tautulli, Trakt)
2. Configure your AI provider (OpenAI, Ollama, LM Studio, or compatible service)
3. (Optional) Set up OAuth for authentication with Google, GitHub, etc.
