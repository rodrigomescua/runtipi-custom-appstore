# Obsidian

Powerful note-taking app for connected thoughts

Obsidian is a personal knowledge management system that helps you capture, organize, and connect your thoughts. With a clean, distraction-free interface and powerful linking capabilities, you can create a web of interconnected notes that grow with your knowledge.

## Features

- **Local Storage**: Keep your notes on your device with full ownership and privacy
- **Powerful Linking**: Create connections between notes for a personal knowledge network
- **Hundreds of Plugins**: Extend functionality with community-created plugins
- **Customizable Themes**: Personalize your note-taking experience with themes
- **Offline Access**: Access your notes offline without internet
- **End-to-End Encryption**: Optional sync with complete encryption
- **GUI Desktop Environment**: Full graphical interface accessible via web browser
- **Multiple Architectures**: Runs on both x86-64 and ARM64 processors

## Security Notes

- **HTTPS Required**: Modern features (WebCodecs) require HTTPS
- **HTTP Basic Auth**: Optional authentication for local networks
- **Privileged Access**: Container has root access within the desktop environment
- **Recommended**: Use a reverse proxy (like SWAG) for internet exposure

## Getting Started

1. Access Obsidian at https://yourhost:8844/
2. Create a new vault or open an existing one
3. Start creating interconnected notes
4. Install plugins and themes from the community
