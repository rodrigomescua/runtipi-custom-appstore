# VCompose - Visual Docker Compose Builder

VCompose is a visual, drag-and-drop interface for creating Docker Compose files. Instead of writing YAML by hand, you can:

- **Drag & drop** services from the sidebar onto a canvas
- **Configure** services with environment variables, ports, and volumes
- **Draw connections** between services to auto-generate `depends_on` configurations
- **Preview & export** production-ready docker-compose.yml files

## Features

- 111+ pre-built services available from the sidebar
- AI-powered composition generation (supports OpenAI, Anthropic, Gemini)
- Real-time YAML preview with syntax highlighting
- Full undo/redo support
- Import existing docker-compose.yml files to visualize them
- Auto-save to browser storage
- No backend required - all processing happens client-side
- Compatible with any container platform (Docker, Podman, etc.)

## Use Cases

- Quickly design multi-service applications without remembering YAML syntax
- Learn Docker Compose through visual interaction
- Generate starter configurations for complex applications
- Share and collaborate on compose configurations
