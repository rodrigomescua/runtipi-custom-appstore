# Piwigo

Piwigo is a powerful open-source photo gallery software designed to help you organise, share and manage your photo collections on the web.

## Features

- **Photo Management**: Upload, organise and manage your photos with ease
- **Album Organisation**: Create albums and sub-albums to categorise your photos
- **User Management**: Control who can access your photos with detailed permission system
- **Tags & Metadata**: Add tags, descriptions and geolocation data to your photos
- **Plugins & Themes**: Extend functionality with hundreds of available plugins and customise appearance with themes
- **Privacy Controls**: Decide who can see which photos with granular privacy settings
- **Multi-language Support**: Available in over 70 languages
- **Mobile Friendly**: Responsive design works on all devices

## Initial Setup

After installation:

1. Open Piwigo in your browser
2. Complete the installation wizard
3. Use the database credentials you configured during installation:
   - **Database host**: `piwigo_db`
   - **Database user**: The user you specified
   - **Database name**: The database name you specified
   - **Database password**: The password you configured
4. Create your admin account
5. Start uploading and organising your photos!

## Default Configuration

- The gallery is accessible via the Runtipi interface
- All photos and configuration are stored in persistent volumes
- MariaDB is used as the database backend
- The LinuxServer.io image provides automatic updates and security patches
