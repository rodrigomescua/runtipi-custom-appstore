# Faved

A lightweight self-hosted web application to store and organize web links with an advanced tagging system.

## Features

- **Clean Interface** – Distraction-free responsive UI for managing bookmarks on desktop and mobile
- **Advanced Tagging System** – Organize bookmarks with colored nested tags, pin important tags for quick access
- **Browser Bookmarklet** – Save bookmarks from any browser without extensions
- **Lightweight & Swift** – Minimal resource usage with fast loading times
- **Import from Browser** – Import bookmarks from Chrome, Safari, Firefox, Edge with folder structure preservation
- **One-Click Migration** – Move saved links, tags, and notes from Pocket
- **Dark Mode** – Sleek dark mode support with responsive design
- **Open Source** – MIT licensed, inspect and contribute to the code

## Getting Started

1. Access the application at the configured URL
2. Create your bookmarks using the simple interface or the browser bookmarklet
3. Organize with nested tags and color coding
4. Use instant search to find bookmarks quickly
5. Import existing bookmarks from your browser or Pocket

## Documentation

For more information, visit the [official documentation](https://faved.dev/docs).

## Browser Bookmarklet

Once installed, you can add a bookmarklet to your browser for quick bookmark saving from any webpage. The setup guide is available in the application settings.
