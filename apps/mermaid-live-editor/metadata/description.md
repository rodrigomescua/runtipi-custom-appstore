# Mermaid Live Editor

A web-based, open-source editor for creating, editing, and sharing Mermaid diagrams.

## Features

- Real-time diagram preview
- Support for multiple diagram types:
  - Flowcharts
  - Sequence diagrams
  - Class diagrams
  - State diagrams
  - Entity relationship diagrams
  - Gantt charts
  - And more
- Simple, intuitive interface
- Export diagrams as images or SVG
- Share diagrams via URL
- Full Mermaid syntax support
- Mobile-friendly interface

## Use Cases

- Technical documentation
- Architecture visualization
- Process flowcharts
- Team planning and communication
- Educational content
