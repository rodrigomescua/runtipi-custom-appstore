# FileGator

FileGator is a self-hosted web file manager for browsing, uploading, downloading, and organizing files through the browser.

## Features

- Multi-user access with roles and permissions
- Drag-and-drop uploads with chunked transfer support
- File operations like copy, move, rename, delete, zip, and unzip
- Repository-backed storage with persistent private user data
- Clean browser UI for day-to-day file management

## Access

After installation, open FileGator on the assigned port and log in with the default credentials:

- Username: `admin`
- Password: `admin123`

For more information, see the [official FileGator documentation](https://docs.filegator.io/).
