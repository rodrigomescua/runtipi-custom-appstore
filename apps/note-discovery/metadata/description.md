## Getting Started

1. Access at `http://localhost:8841` (or your configured domain)
2. Login with your configured password
3. Start creating and discovering notes!

Default volumes:
- `/app/data` - Your personal notes (plain markdown files)
- `/app/plugins` - Plugin directory
- `/app/themes` - Theme files (CSS)
- `/app/config.yaml` - Configuration file

For detailed documentation, visit: https://github.com/gamosoft/NoteDiscovery
