# Degoog

Degoog is a self-hosted meta search engine that aggregates results from multiple search engines while keeping your searches private. It runs entirely on your own infrastructure, with no tracking or data collection.

## Features

- **Multi-engine search**: Aggregates results from Google, Bing, DuckDuckGo, and others
- **Privacy-focused**: No logs, no tracking, no profiling
- **Extensible**: Plugin system for custom engines, themes, autocomplete providers, and more
- **Built-in indexer**: Stores and re-ranks your past search results for faster, personalized results
- **Settings protection**: Optional password to secure the configuration panel
- **Public or private**: Can be run as a personal instance or opened to the public

## Usage

After installation, access the interface via the assigned port. Optionally set a settings password to prevent unauthorized configuration changes.

For advanced configuration and environment variables, refer to the [official documentation](https://degoog-org.github.io/docs/env.html).
