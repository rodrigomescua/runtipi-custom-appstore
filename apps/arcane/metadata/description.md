# Arcane

Modern Docker Management, Made for Everyone

Arcane is a powerful yet user-friendly Docker management platform that brings simplicity to container orchestration. With an intuitive web interface, manage all your Docker resources seamlessly:

- **Containers**: Deploy, monitor, and manage containers with real-time statistics
- **Images**: Pull, build, and manage Docker images with version tracking
- **Networks**: Create and configure Docker networks for your applications
- **Volumes**: Manage persistent data storage effortlessly
- **Projects**: Organize and deploy Docker Compose projects directly from the UI
- **Auto-Updates**: Keep your containers updated automatically with smart detection
- **Multiple Environments**: Manage local and remote Docker environments from one place
- **GPU Monitoring**: Monitor GPU usage in real-time (NVIDIA support)
- **Notifications**: Get alerts for container and image updates via multiple channels
- **Security**: Built with OIDC authentication and encryption support

Arcane makes Docker management accessible to everyone, from beginners to advanced users.
