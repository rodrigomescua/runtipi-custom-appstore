## Donetick

Donetick is an open-source, user-friendly app designed to help you organize tasks and chores effectively featuring customizable options to help you and others stay organized.

### Major Features Include

- **Collaborative Task Management**: Create and manage tasks solo or with family and friends. Share or assign tasks and chores with others in a group.
- **Natural Language Task Creation**: Describe what you need to do in plain English. Donetick automatically extracts dates, times, and recurrence patterns.
- **Advanced Task Scheduling**: Supports flexible scheduling (daily, weekly, monthly, yearly) with assignee rotation and time tracking.
- **Subtasks with Smart Reset**: Break tasks into smaller steps with automatic reset for recurring tasks.
- **Organize with Priorities and Labels**: Use custom labels and five priority levels (P1-P4, No Priority) to stay organized.
- **Add Photos**: Attach photos directly to tasks with support for cloud storage (AWS S3, Cloudflare R2, MinIO).
- **Things Tracking**: Track data that isn't a task - numbers, booleans, or text.
- **NFC Tag Support**: Create physical triggers to instantly mark tasks as complete.
- **Points System**: Built-in gamification to reward task completion and track progress.
- **Comprehensive Analytics**: View task breakdowns by label, completion status, and helpful graphs.
- **Multi-Factor Authentication**: Support for TOTP-based MFA.
- **Multiple Sign-In Options**: Local accounts or OAuth2 providers (Keycloak, Authentik, Authelia).
- **Dashboard View**: Mount-friendly dashboard layout for larger screens.
- **Realtime Sync**: Instantly reflect task changes across all connected devices and users.
- **Offline Support**: Limited functionality available without connection.
- **Multi-Platform Notifications**: Get reminders via mobile app, Telegram, Discord, or Pushover.
- **Home Assistant Integration**: Manage tasks within Home Assistant.
- **REST API**: Full access through REST API for custom automations.
- **Webhook System**: Connect to external systems with flexible webhook support.
